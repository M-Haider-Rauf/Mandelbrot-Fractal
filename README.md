# Mandelbrot-Fractal
This is a mandelbrot fractal plotter written in C++ using the SDL2 graphics library. The mandelbrot set is one of the most intriguing mathematical
concepts you'll ever see.

# Key bindings
* **WASD** to pan around
* **+/-** to zoom in/out
* **C** to capture screenshot
